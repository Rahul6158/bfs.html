Breadth-First Search (BFS) in Data Structures

What is Breadth-First Search (BFS)?

Breadth-First Search (BFS) is an algorithm for traversing or searching tree or graph data structures. It explores all the vertices (or nodes) of a graph or tree level by level, starting from a given node (called the source or root). BFS uses a queue to keep track of nodes to explore next.

Key Concepts of BFS

  • Level-by-Level Traversal: BFS starts at the source node and explores all its neighboring nodes first, before moving on to the next level.
  • Queue: BFS uses a queue data structure to keep track of nodes to visit. The queue follows the First In, First Out (FIFO) principle, ensuring that nodes are processed in the correct order.
  • Unweighted Shortest Path: BFS can be used to find the shortest path between two nodes in an unweighted graph because it explores nodes in the order of their distance from the source.

How BFS Works

  1. Start from the Source Node: The algorithm starts from a given node and enqueues it (adds it to the queue).
  2. Visit Neighbors: For the node at the front of the queue, visit all its unvisited neighbors and enqueue them.
  3. Dequeue and Repeat: Dequeue the current node, then repeat the process for the next node in the queue until all nodes are visited.

BFS Algorithm

  1. Initialization:
    • Mark all nodes as unvisited.
    • Add the source node to the queue and mark it as visited.
  2. While the queue is not empty:
    • Dequeue the front node.
    • Visit all its adjacent nodes. If a node has not been visited, mark it as visited and enqueue it.
  3. Repeat until the queue is empty.

BFS Example (Graph)

Consider the following graph: 

    A
   / \
  B   C
 / \   \
D   E   F

- Start from node A. - BFS first visits all nodes directly connected to A: nodes B and C. - Then, it visits all nodes connected to B and C, i.e., nodes D, E, and F.

The BFS traversal order for this graph is: A → B → C → D → E → F.

Applications of BFS

  • Shortest Path in an Unweighted Graph: BFS is used to find the shortest path from the source node to any other node.
  • Web Crawlers: Search engines use BFS to explore all the links on a web page.
  • Social Networks: BFS can be used to find the shortest degree of separation between people (e.g., friends-of-friends).
  • Solving Puzzles and Games: BFS can explore possible moves in puzzles or games (like solving a maze).
